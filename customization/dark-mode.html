Dark mode

This document provides detailed instructions on how to enable the dark mode feature for Biel.ai on your website. By following these steps, you can enhance the visual experience for users who prefer a darker color scheme.

Enable the dark mode​

To activate dark mode for Biel.ai, you will need to modify the HTML structure of your website.

This involves adding a specific attribute to the main <html> tag of your site's HTML code.

<html data-theme="dark">
<!-- Your existing HTML content -->
</html>
