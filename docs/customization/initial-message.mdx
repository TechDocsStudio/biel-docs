# Initial message

The initial message is the first text the chatbot displays when a user opens it. Use it to greet visitors, provide instructions, or set the tone.

:::important
Only **Administrator** or **Maintainer** roles can manage projects. See [Manage roles](../administration/roles.md).
:::

## Customize the initial message

1. In the [Biel.ai dashboard](https://app.biel.ai), select your project.
2. Click **Settings**.
3. Under **UI settings > Initial message**, enter the text you want to display.
4. Click **Save**.

Open the chatbot on your site to verify the updated message appears.
