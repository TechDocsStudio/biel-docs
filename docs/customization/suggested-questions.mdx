# Suggested questions

Suggested questions are clickable prompts that appear when a user opens the chatbot. Use them to surface common queries so users don't have to type from scratch.

:::important
Only **Administrator** or **Maintainer** roles can manage projects. See [Manage roles](../administration/roles.md).
:::

## Add suggested questions

1. In the [Biel.ai dashboard](https://app.biel.ai), select your project.
2. Click **Settings**.
3. Under **UI settings > Suggested questions**, enter your questions. Click **Add question** to add more.
4. Click **Save**.

Open the chatbot on your site to verify the suggested questions appear as clickable options.
