# Text

Biel.ai provides comprehensive text customization options, allowing you to align the chatbot's textual content with your company's tone or support various languages. By customizing these properties, you can ensure a consistent user experience that reflects your brand's voice or caters to your diverse user base.

## Usage

To customize the textual properties, set the respective attributes on the `<biel-button>` element in your HTML. Here's an illustrative example:

```html
<biel-button 
  project="YOUR_PROJECT_ID"
  footer-text="This content is generated by AI.">
  Button text
</biel-button>
```

## Properties

The table below lists all the available properties you can override:

| Property                       | Description                                                                                                                     |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------- |
| `error-message-4-0-3`          | **Type**: `string`<br />**Default value**: `Oops! The request URL does not match the one defined for this project.`<br />403 unauthorized message.<br /><br />    |
| `error-message-4-0-4`          | **Type**: `string`<br />**Default value**: `Oops! We could not find the provided project ID.`<br />404 not found message.<br /><br />    |
| `error-message-default`        | **Type**: `string`<br />**Default value**: `Oops! Please try again later.<br />500 error message.`<br /><br />    |
| `footer-text`                  | **Type**: `string`<br />**Default value**: `This AI is experimental. Always verify the answers before using them.`<br />Footer text<br /><br />    |
