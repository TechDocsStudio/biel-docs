# Layout

The Biel.ai chatbot is highly customizable, allowing you to customize its behavior to suit your application's needs.
This page lists all available properties you can use to adjust the behavior and appearance of the Biel.ai button in your application.

### Usage

To customize the properties, you need to set the appropriate attributes on the `<biel-button>` element in your HTML. For example:

```html
<biel-button project="YOUR_PROJECT_ID" button-style="dark">Ask AI</biel-button>
```

## Properties
The table below lists all the available properties that modify the layout:

| Property               | Description                                                                                   |
|------------------------|-----------------------------------------------------------------------------------------------|
| `project`              | **Required**<br />**Type**: `string`<br /><br />The project ID.         |
| `button-position`      | **Type**: `string`<br />**Default value**: `default`<br />**Available values:** `default`, `bottom-right`, `center-right`<br /><br />Position of the button.|
| `button-style`         | **Type**: `string`<br />**Default value**: `default`<br />**Available values:** `default`, `dark`, `light`<br /><br />Style of the button.|
| `custom-font`          | **Type**: `boolean`<br />**Default value**: `false`<br /><br />When set to `true`, the [Inter font](https://fonts.google.com/specimen/Inter) will not be loaded. Useful if your site already uses a different font, enabling a quicker load time.|
| `expand-modal`         | **Type**: `boolean`<br />**Default value**: `false`<br /><br />When set to `true`, the modal opens expanded. |
| `hide-icon`            | **Type**: `boolean`<br />**Default value**: `false`<br /><br />When set to `true`, the button icon hides. |
| `hide-expand-button`   | **Type**: `boolean`<br />**Default value**: `false`<br /><br />When set to `true`, the expand button icon hides. |
| `modal-position`       | **Type**: `string`<br />**Default value**: `default`<br />**Available values:** `center`, `bottom-right`, `bottom-left`<br /><br />Position of the modal.|
| `error-message-4-0-3`          | **Type**: `string`<br />**Default value**: `Oops! The request URL does not match the one defined for this project.`<br />403 unauthorized message.<br /><br />    |
| `error-message-4-0-4`          | **Type**: `string`<br />**Default value**: `Oops! We could not find the provided project ID.`<br />404 not found message.<br /><br />    |
| `error-message-default`        | **Type**: `string`<br />**Default value**: `Oops! Please try again later.<br />500 error message.`<br /><br />    |
| `footer-text`                  | **Type**: `string`<br />**Default value**: `This AI is experimental. Always verify the answers before using them.`<br />Footer text<br /><br />    |
