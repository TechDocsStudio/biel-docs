# Styles

The Biel.ai button is highly customizable, allowing you to customize its appearance to suit your application's needs.
This reference document lists all the available CSS custom properties that can be overridden to match your website's design.

## Usage

To customize these properties, define them in your website's CSS using the `:root` selector for global changes or a specific selector to target particular areas:

```css
:root {
    --biel-primary-color: #00D68F;
    ...
}
```

## Properties

The tables below lists all the available properties you can override:

```css
:root {
    /* Colors */
    --biel-primary-color: #0070F4;
    --biel-secondary-color: #f1f3f4;
    --biel-light-color: #ccc;
    --biel-dark-color: #191919;
    --biel-text-color: #5f6368;
    --biel-white-color: #fff;
    --biel-highlight-color: #ffb422;
       
    /* Fonts */
    --biel-font-family: "Inter", "Tahoma", "sans-serif";
    --biel-heading-font-size: 16px;
    --biel-text-font-size: 14px;

    /* Biel button*/
    --biel-button-border-radius: 20px;
    --biel-button-dark-bg-color: var(--biel-primary-color);
    --biel-button-dark-text-color: var(--biel-white-color);
    --biel-button-light-bg-color: var(--biel-white-color);
    --biel-button-light-text-color: var(--biel-primary-color);
    --biel-button-text-font-size: var(--biel-text-font-size);
    --biel-button-text-font-weight: 600;
    --biel-button-z-index: 300;

    
    /*Biel search*/
    --biel-search-header-font-family: var(--biel-font-family);
    --biel-search-header-font-size: var(--biel-heading-font-size);
    --biel-search-header-font-weight: 500;
    --biel-search-header-text-color: var(--biel-dark-color);
    --biel-search-close-bg-color: var(--biel-white-color);
    --biel-search-close-color: var(--biel-dark-color);
    --biel-search-content-bg-color: var(--biel-white-color);
    --biel-search-content-border-color: var(--biel-light-color);
    --biel-search-content-border-radius: 8px;
    --biel-search-content-font-family: var(--biel-font-family);
    --biel-search-content-position-bottom: 10px;
    --biel-search-content-position-left: 10px;
    --biel-search-content-position-right: 10px;
    --biel-search-content-position-top: 10px;
    --biel-search-content-text-color: var(--biel-text-color);
    --biel-search-content-z-index: 300;
    --biel-search-footer-link: var(--biel-dark-color);
}

[data-theme='dark'] {
    --biel-text-color: #b0b8c4;
    --biel-light-color: #363636;
    --biel-search-button-bg-color-active: transparent;
    --biel-search-button-icon-color: var(--biel-white-color);
    --biel-search-button-text-color: var(--biel-white-color);
    --biel-search-header-text-color: var(--biel-white-color);
    --biel-search-close-bg-color: transparent;
    --biel-search-close-color: var(--biel-white-color);
    --biel-search-content-bg-color: var(--biel-dark-color);
    --biel-search-content-text-color: var(--biel-text-color);
    --biel-search-footer-link: var(--biel-white-color);
}  
```
