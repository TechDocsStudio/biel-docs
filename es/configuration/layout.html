Layout

The Biel.ai button is highly customizable, allowing you to customize its behavior to suit your application's needs. This page lists all available properties you can use to adjust the behavior and appearance of the Biel.ai button in your application.

Usage​

To customize the properties, you need to set the appropriate attributes on the <biel-button> element in your HTML. For example:

<biel-button project="YOUR_PROJECT_ID" button-style="dark" hide-icon="true">Ask AI</biel-button>

Properties​

The table below lists all the available properties you can set:

PropertyDescription
projectRequired
Type: string

The project ID.
button-positionType: string
Default value: default
Available values: default, bottom-right, center-right

Position of the button.
button-styleType: string
Default value: default
Available values: default, dark, light

Style of the button.
custom-fontType: boolean
Default value: false

When set to true, the Inter font will not be loaded. Useful if your site already uses a different font, enabling a quicker load time.
